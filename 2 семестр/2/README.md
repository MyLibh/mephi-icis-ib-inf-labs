# Лабораторная работа №2.15

## Задание

Из входного потока вводится непрямоугольная матрица целых чисел **[a_{ij}]**, __i__=1,...,__n__, __j__=1,...,__m_i__.

Значения __n__ и __m_i__ заранее не известны и вводятся из входного потока.

Сформировать новую матрицу, поместив в ее **i**-ую строку только те элементы из **i**-ой строки исходной матрицы,
которые встречаются в ней 2 и более раз.

В дополнительных функциях доступ к элементам массивов реализовать с помощью указателей.
Исходную и полученную матрицу вывести в выходной поток с необходимыми комментариями.

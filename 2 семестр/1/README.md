# Лабораторная работа №1.15

## Задание

Дана последовательность из __n__ натуральных чисел.

Для каждого числа исходной последовательности увеличить каждую цифру в записи числа на единицу (вместо цифры 9 записать 0)
и подсчитать количество цифр в записи числа.

Сформировать новую последовательность, в которой сначала расположить числа исходной последовательности в порядке возрастания
количества цифр в записи числа, а затем преобразованные числа в порядке убывания их значений. Вывести исходную и полученную 
последовательность на экран.

В программе использовать функцию, которая увеличивает каждую цифру в записи числа на единицу и возвращает количество цифр в записи числа.
